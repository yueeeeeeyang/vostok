🌐 Vostok.Web

Web 服务器模块

基于 Java NIO Reactor 模式的高性能 HTTP 服务器，支持路由、中间件、WebSocket、SSE、静态资源、GZIP、CORS，以及根据实体类自动生成 REST CRUD API。

初始化与启动

Vostok.Web.init(new VKWebConfig()
    .port(8080)
    .ioThreads(2)
    .workerThreads(16)
);

// 注册路由
Vostok.Web.get("/hello", (req, res) -> res.ok("Hello!"));
Vostok.Web.post("/users", (req, res) -> {
    String body = req.body();
    res.status(201).json("{\"created\":true}");
});

Vostok.Web.start();

路由

基本路由

// GET / POST / PUT / DELETE / PATCH
Vostok.Web.get("/items", handler);
Vostok.Web.post("/items", handler);
Vostok.Web.put("/items/{id}", handler);
Vostok.Web.delete("/items/{id}", handler);

// 路径参数
Vostok.Web.get("/users/{id}", (req, res) -> {
    String id = req.pathParam("id");
    res.json("{\"id\":" + id + "}");
});

// 查询参数
Vostok.Web.get("/search", (req, res) -> {
    String q = req.queryParam("q");
    int page = req.queryParamInt("page", 1);
    res.ok("q=" + q + " page=" + page);
});

中间件

// 全局中间件
Vostok.Web.use((req, res, next) -> {
    String token = req.header("Authorization");
    if (token == null) { res.status(401).json("{\"error\":\"unauthorized\"}"); return; }
    next.handle();  // 放行
});

CORS

// 快捷开启（允许所有来源）
Vostok.Web.cors();

// 自定义
Vostok.Web.cors(new VKCorsConfig()
    .allowedOrigins("https://example.com")
    .allowedMethods("GET", "POST")
    .allowCredentials(true));

GZIP

Vostok.Web.gzip();  // 默认配置（>1KB 自动压缩）
// 或自定义阈值
Vostok.Web.gzip(new VKGzipConfig().minSizeBytes(512));

限流

// 全局限流（每秒最多 1000 请求）
Vostok.Web.rateLimit(new VKRateLimitConfig().qps(1000));

// 单路由限流
Vostok.Web.rateLimit(VKHttpMethod.POST, "/login",
    new VKRateLimitConfig().qps(10).rejectStatus(429));

静态资源

Vostok.Web.staticDir("/static", "./public");

自动 CRUD API

扫描 @VKEntity 实体类，自动为每个实体生成标准 REST 接口。

// 扫描默认包
Vostok.Web.autoCrudApi();

// 指定包 + RESTful 风格
Vostok.Web.autoCrudApi(VKCrudStyle.RESTFUL, "com.example.entity");

例如 User 实体，自动生成：

  • GET /users — 查询全部
  • GET /users/{id} — 按主键查询
  • POST /users — 新增
  • PUT /users/{id} — 更新
  • DELETE /users/{id} — 删除

WebSocket

Vostok.Web.websocket("/ws/chat", new VKWebSocketHandler() {
    public void onOpen(String sessionId) {
        System.out.println("connected: " + sessionId);
    }
    public void onMessage(String sessionId, String text) {
        // 广播给同 path 所有连接
        Vostok.Web.websocketBroadcast("/ws/chat", text);
    }
    public void onClose(String sessionId) {}
});

// 按房间广播
Vostok.Web.websocketBroadcastRoom("/ws/chat", "room-1", "hello room");

// 按分组广播
Vostok.Web.websocketBroadcastGroup("/ws/chat", "vip", "vip msg");

Server-Sent Events (SSE)

Vostok.Web.sse("/events", (req, emitter) -> {
    emitter.send("data: connected\n\n");
    // emitter.close() 关闭流
});

健康检查 / 指标

Vostok.Web.health();           // GET /health
Vostok.Web.health("/_ping");  // 自定义路径
Vostok.Web.metrics();          // GET /metrics

错误处理

Vostok.Web.error((req, res, ex) -> {
    res.status(500).json("{\"error\":\"" + ex.getMessage() + "\"}");
});
