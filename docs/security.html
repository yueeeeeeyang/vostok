🔒 Vostok.Security

Security 安全模块

主动调用型安全工具集，提供 SQL 注入、XSS、命令注入、路径穿越、SSRF、XXE、CRLF、NoSQL 注入检测，响应脱敏，文件类型验证，以及 AES/RSA 加解密和 Key Store 管理。

注意
Security 模块为主动调用型，不会自动接入 Data/Web 执行链路，需要在业务代码中显式调用检测方法。

初始化

// 默认配置
Vostok.Security.init();

// 严格模式
Vostok.Security.init(new VKSecurityConfig()
    .strictMode(true)
    .auditLogEnabled(true)
);

SQL 注入检测

// 检测用户输入
boolean safe = Vostok.Security.checkSql(userInput);
if (!safe) {
    throw new SecurityException("SQL injection detected");
}

// 自定义规则
Vostok.Security.registerRule(input -> !input.contains("EXEC"));
List<String> rules = Vostok.Security.listRules();

XSS 检测与净化

// 检测 XSS
boolean safe = Vostok.Security.checkXss(input);

// HTML 净化（转义危险标签）
String clean = Vostok.Security.sanitizeHtml(userHtml);

命令注入检测

boolean safe = Vostok.Security.checkCommandInjection(input);

路径穿越检测

boolean safe = Vostok.Security.checkPathTraversal(filePath);

SSRF 检测

boolean safe = Vostok.Security.checkSsrf(url);

XXE 检测

boolean safe = Vostok.Security.checkXxe(xmlContent);

CRLF 注入检测

boolean safe = Vostok.Security.checkCrlf(headerValue);

NoSQL 注入检测

boolean safe = Vostok.Security.checkNoSqlInjection(mongoQuery);

文件类型检测

// 通过魔数检测文件类型（防止伪装后缀上传）
byte[] fileBytes = req.bodyBytes();
boolean valid = Vostok.Security.checkFile(fileBytes);

// 通过文件路径检测
boolean valid = Vostok.Security.checkFile(Path.of("upload/photo.jpg"));

响应脱敏

// 对 JSON 响应中的敏感字段自动打码
String safeJson = Vostok.Security.desensitizeResponse(
    responseJson,
    Set.of("password", "idCard", "phone", "creditCard")
);
// 输出示例：{"phone":"138****5678","password":"****"}

AES 加解密

// 生成 AES 密钥（Base64 编码）
String keyBase64 = Vostok.Security.AesCrypto.generateKey();

// 加密
String cipher = Vostok.Security.AesCrypto.encrypt("hello", keyBase64);

// 解密
String plain = Vostok.Security.AesCrypto.decrypt(cipher, keyBase64);

RSA 加解密

// 生成 RSA 密钥对
VKRsaKeyPair pair = Vostok.Security.RsaCrypto.generateKeyPair();
String pubKey  = pair.publicKeyBase64();
String privKey = pair.privateKeyBase64();

// 公钥加密
String cipher = Vostok.Security.RsaCrypto.encrypt("hello", pubKey);

// 私钥解密
String plain = Vostok.Security.RsaCrypto.decrypt(cipher, privKey);

Hash 与 HMAC

// SHA-256 哈希
String hash = Vostok.Security.HashCrypto.sha256("password");

// HMAC-SHA256
String secretBase64 = Vostok.Security.HashCrypto.generateHmacKey();
String mac = Vostok.Security.HashCrypto.hmacSha256("message", secretBase64);

Key Store（密钥存储）

// 初始化本地密钥存储
Vostok.Security.initKeyStore(new VKKeyStoreConfig()
    .dir("./keys")
    .masterPassword("masterSecret"));

// 保存密钥（带 TTL）
Vostok.Security.saveKey("aes-key-v1", keyBytes, 86400_000);

// 加载密钥
byte[] k = Vostok.Security.loadKey("aes-key-v1");

// 管理
Vostok.Security.deleteKey("aes-key-v1");
List<String> ids = Vostok.Security.listKeyIds();

// 注册自定义 Key Store
Vostok.Security.setKeyStore(myRedisKeyStore);

API 速查

方法说明
checkSql(input)SQL 注入检测，安全返回 true
checkXss(input)XSS 检测
sanitizeHtml(html)HTML 净化/转义
checkCommandInjection(input)命令注入检测
checkPathTraversal(path)路径穿越检测
checkSsrf(url)SSRF 检测
checkXxe(xml)XXE 检测
checkCrlf(input)CRLF 注入检测
checkNoSqlInjection(input)NoSQL 注入检测
checkFile(bytes)文件魔数类型验证
desensitizeResponse(json, fields)响应脱敏（按字段名打码）
AesCrypto.encrypt(plain, keyB64)AES 加密
AesCrypto.decrypt(cipher, keyB64)AES 解密
RsaCrypto.generateKeyPair()生成 RSA 密钥对
RsaCrypto.encrypt(plain, pubKey)RSA 公钥加密
RsaCrypto.decrypt(cipher, privKey)RSA 私钥解密
HashCrypto.sha256(input)SHA-256 哈希
HashCrypto.hmacSha256(input, key)HMAC-SHA256
registerRule(rule)注册自定义 SQL 安全规则
saveKey(id, bytes, ttlMs)保存密钥到 Key Store
loadKey(id)从 Key Store 加载密钥
