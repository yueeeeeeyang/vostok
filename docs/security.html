🔒 Vostok.Security

Security 安全模块

主动调用型安全工具集，提供 SQL 注入、XSS、命令注入、路径穿越、SSRF、XXE、CRLF、NoSQL 注入检测，响应脱敏，文件类型验证，以及 AES/RSA 加解密和 Key Store 管理。

注意
Security 模块为主动调用型，不会自动接入 Data/Web 执行链路，需要在业务代码中显式调用检测方法。

初始化

// 默认配置
Vostok.Security.init();

// 严格模式
Vostok.Security.init(new VKSecurityConfig()
    .strictMode(true)
    .auditLog(true)
);

SQL 注入检测

// 检测用户输入
VKSqlCheckResult result = Vostok.Security.checkSql(userInput);
if (!result.isSafe()) {
    throw new SecurityException("SQL injection detected: " + result.getReason());
}

// 便捷方法（直接返回 boolean）
boolean safe = Vostok.Security.isSafeSql(userInput);

// 断言式（不安全时直接抛异常）
Vostok.Security.assertSafeSql(userInput);

// 自定义规则
Vostok.Security.registerRule(input -> !input.contains("EXEC"));
List<String> rules = Vostok.Security.listRules();

XSS 检测与净化

// 检测 XSS
VKSecurityCheckResult r = Vostok.Security.checkXss(input);
if (!r.isSafe()) { ... }

// XSS 净化（转义危险标签）
String clean = Vostok.Security.sanitizeXss(userHtml);

命令注入检测

VKSecurityCheckResult r = Vostok.Security.checkCommandInjection(input);
if (!r.isSafe()) { ... }

路径穿越检测

VKSecurityCheckResult r = Vostok.Security.checkPathTraversal(filePath);
if (!r.isSafe()) { ... }

SSRF 检测

VKSecurityCheckResult r = Vostok.Security.checkSsrf(url);
if (!r.isSafe()) { ... }

XXE 检测

VKSecurityCheckResult r = Vostok.Security.checkXxe(xmlContent);
if (!r.isSafe()) { ... }

CRLF 注入检测

VKSecurityCheckResult r = Vostok.Security.checkCrlf(headerValue);
if (!r.isSafe()) { ... }

NoSQL 注入检测

VKSecurityCheckResult r = Vostok.Security.checkNoSqlInjection(mongoQuery);
if (!r.isSafe()) { ... }

文件类型检测

// 检测文件魔数类型
VKFileType type = Vostok.Security.detectFileType(fileBytes);

// 校验是否为允许的类型（防伪装后缀上传）
boolean valid = Vostok.Security.checkFileMagic(fileBytes, VKFileType.JPEG, VKFileType.PNG);

// 检测是否为可执行脚本（上传安全）
boolean dangerous = Vostok.Security.checkExecutableScriptUpload(fileName, fileBytes);

响应脱敏

// 对 JSON 响应中的敏感字段自动打码
String safeJson = Vostok.Security.maskSensitiveResponse(responseJson);

// 检测响应是否包含敏感信息
VKSecurityCheckResult r = Vostok.Security.checkSensitiveResponse(responseJson);

AES 加解密

// 生成 AES 密钥（Base64 编码）
String keyBase64 = Vostok.Security.generateAesKey();

// 加密
String cipher = Vostok.Security.encrypt("hello", keyBase64);

// 解密
String plain = Vostok.Security.decrypt(cipher, keyBase64);

RSA 加解密

// 生成 RSA 密钥对
VKRsaKeyPair pair = Vostok.Security.generateRsaKeyPair();
String pubKeyPem  = pair.publicKeyPem();
String privKeyPem = pair.privateKeyPem();

// 公钥加密
String cipher = Vostok.Security.encryptByPublicKey("hello", pubKeyPem);

// 私钥解密
String plain = Vostok.Security.decryptByPrivateKey(cipher, privKeyPem);

// 签名验证
String sig = Vostok.Security.sign("message", privKeyPem);
boolean ok = Vostok.Security.verify("message", sig, pubKeyPem);

Hash 与 HMAC

// SHA-256 哈希
String hash = Vostok.Security.sha256Hex("password");

// HMAC-SHA256
String mac = Vostok.Security.hmacSha256("message", secretBase64);

Key Store（密钥存储）

// 初始化 Key Store
Vostok.Security.initKeyStore(new VKKeyStoreConfig()
    .dir("./keys")
    .masterPassword("masterSecret"));

// 按 keyId 自动创建/获取 AES 密钥
byte[] aesKey = Vostok.Security.getOrCreateAesKey("aes-key-v1");

// 带 TTL（秒）
byte[] aesKey = Vostok.Security.getOrCreateAesKey("aes-key-v1", 86400L);

// 按 keyId 自动创建/获取 RSA 密钥对
VKRsaKeyPair pair = Vostok.Security.getOrCreateRsaKeyPair("rsa-key-v1");

// 使用 keyId 加解密（自动密钥管理）
String cipherPayload = Vostok.Security.encryptWithKeyId("hello", "aes-key-v1");
String plain = Vostok.Security.decryptWithKeyId(cipherPayload);

// 密钥轮转
Vostok.Security.rotateAesKey("aes-key-v1");
Vostok.Security.rotateRsaKeyPair("rsa-key-v1");
Vostok.Security.rotateKek("aes-key-v1");

API 速查

方法说明
checkSql(input)SQL 注入检测，返回 VKSqlCheckResult；isSafeSql(input) 返回 boolean
checkXss(input)XSS 检测，返回 VKSecurityCheckResult
sanitizeXss(html)XSS 净化/转义
checkCommandInjection(input)命令注入检测，返回 VKSecurityCheckResult
checkPathTraversal(path)路径穿越检测，返回 VKSecurityCheckResult
checkSsrf(url)SSRF 检测，返回 VKSecurityCheckResult
checkXxe(xml)XXE 检测，返回 VKSecurityCheckResult
checkCrlf(input)CRLF 注入检测，返回 VKSecurityCheckResult
checkNoSqlInjection(input)NoSQL 注入检测，返回 VKSecurityCheckResult
detectFileType(bytes)文件魔数类型检测
checkFileMagic(bytes, types...)校验允许的文件类型
maskSensitiveResponse(json)响应敏感信息打码
encrypt(plain, keyB64)AES 加密
decrypt(cipher, keyB64)AES 解密
generateAesKey()生成 AES 密钥（Base64 编码）
generateRsaKeyPair()生成 RSA 密钥对
encryptByPublicKey(plain, pubKey)RSA 公钥加密
decryptByPrivateKey(cipher, privKey)RSA 私钥解密
sign(text, privKey)RSA 签名
verify(text, sig, pubKey)RSA 签名验证
sha256Hex(input)SHA-256 哈希
hmacSha256(input, key)HMAC-SHA256
registerRule(rule)注册自定义 SQL 安全规则
getOrCreateAesKey(keyId)按 keyId 自动创建/获取 AES 密钥
getOrCreateRsaKeyPair(keyId)按 keyId 自动创建/获取 RSA 密钥对
encryptWithKeyId(text, keyId)使用 keyId 加密（自动密钥管理）
decryptWithKeyId(payload)使用 keyId 解密（自动密钥管理）
rotateAesKey(keyId)轮转 AES 密钥
