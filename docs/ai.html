🤖 Vostok.AI

AI 集成模块

统一 AI 客户端，支持多模型注册与切换、会话管理（多轮对话）、JSON 结构化响应、流式输出、RAG 检索增强生成、向量检索、Tool Call 函数调用。

初始化

Vostok.AI.init(new VKAiConfig());

// 注册模型
Vostok.AI.registerModel("gpt4", new VKAiModelConfig()
    .modelType(VKAiModelType.GPT4)
    .apiKey(System.getenv("OPENAI_KEY"))
    .baseUrl("https://api.openai.com/v1")
    .maxTokens(4096)
    .temperature(0.7)
);

Vostok.AI.registerModel("claude", new VKAiModelConfig()
    .modelType(VKAiModelType.CLAUDE)
    .apiKey(System.getenv("ANTHROPIC_KEY"))
);

单轮对话

// 简单 Chat
VKAiChatRequest req = VKAiChatRequest.builder()
    .model("gpt4")
    .userMessage("用一句话解释量子纠缠")
    .systemMessage("你是一位物理学教授")
    .build();

VKAiChatResponse res = Vostok.AI.chat(req);
System.out.println(res.content());

// 异步
CompletableFuture<VKAiChatResponse> future = Vostok.AI.chatAsync(req);

JSON 结构化输出

record Sentiment(String label, double score) {}

Sentiment result = Vostok.AI.chatJson(
    VKAiChatRequest.builder()
        .model("gpt4")
        .userMessage("分析这段评论的情感：'这个产品太棒了！'")
        .systemMessage("返回 JSON: {\"label\": ..., \"score\": ...}")
        .build(),
    Sentiment.class
);

多轮会话

// 创建会话（维护对话历史）
VKAiSession session = Vostok.AI.createSession("gpt4");
String sessionId = session.sessionId();

// 基于会话的对话（自动携带历史）
String reply1 = Vostok.AI.chatSession(sessionId, "Java 有哪些主要特性？");
String reply2 = Vostok.AI.chatSession(sessionId, "它与 Go 有什么区别？");

// 查看历史消息
List<VKAiSessionMessage> history = Vostok.AI.sessionMessages(sessionId);

// 切换模型（保留历史）
Vostok.AI.switchSessionModel(sessionId, "claude");

// 删除会话
Vostok.AI.deleteSession(sessionId);

RAG（检索增强生成）

// 1. 写入文档到 RAG 知识库
Vostok.AI.ingestRagDocument(
    VKAiRagIngestRequest.builder()
        .embeddingModel("text-embedding-ada-002")
        .docId("vostok-readme")
        .content(readmeText)
        .chunkSize(512)
        .build()
);

// 2. RAG 查询
VKAiRagResponse ragRes = Vostok.AI.rag(
    VKAiRagRequest.builder()
        .model("gpt4")
        .embeddingModel("text-embedding-ada-002")
        .query("Vostok 支持哪些缓存后端？")
        .topK(3)
        .build()
);
System.out.println(ragRes.answer());
System.out.println(ragRes.sources());

Tool Call（函数调用）

// 注册工具函数
Vostok.AI.registerTool(VKAiTool.builder()
    .name("get_weather")
    .description("获取指定城市的实时天气")
    .parameter("city", "string", "城市名称", true)
    .handler(args -> fetchWeather(args.get("city")))
    .build());

// 发起支持工具调用的对话
VKAiChatResponse res = Vostok.AI.chat(
    VKAiChatRequest.builder()
        .model("gpt4")
        .userMessage("北京今天天气怎么样？")
        .enableTools(true)
        .build()
);

API 速查

方法说明
registerModel(name, config)注册命名 AI 模型
chat(request)同步单轮对话
chatAsync(request)异步单轮对话
chatJson(request, Class)对话并解析 JSON 结构
chatJsonAsync(request, Class)异步对话解析 JSON
createSession(model)创建多轮对话 Session
chatSession(sessionId, text)基于 Session 的对话
sessionMessages(sessionId)获取会话历史消息
switchSessionModel(sessionId, model)切换会话使用的模型
deleteSession(sessionId)删除会话
embed(request)生成文本 Embedding 向量
rerank(request)文档重排序
setVectorStore(store)设置向量数据库后端
upsertVectorDocs(docs)写入向量文档
searchVector(vector, topK)向量相似度检索
searchKeywords(query, topK)关键词检索
ingestRagDocument(request)向 RAG 知识库写入文档
rag(request)RAG 检索增强对话
registerTool(tool)注册 Tool Call 函数
clearTools()清除所有注册工具
setMemoryStore(store)自定义 Session 持久化存储
