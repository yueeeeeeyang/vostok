📁 Vostok.File

File 文件存储模块

统一文件访问抽象层，默认实现为本地文件系统，支持读写、目录管理、压缩解压、文件监听，可扩展为对象存储（S3/OSS）等后端。

初始化

Vostok.File.init(new VKFileConfig()
    .baseDir("./data/files")
    .charset("UTF-8")
);

文件读写

// 创建文件
Vostok.File.create("config/app.conf", "key=value");

// 覆盖写入
Vostok.File.write("data/report.txt", content);

// 追加写入
Vostok.File.append("logs/access.log", "2024-01-01 GET /\n");

// 读取文本
String text = Vostok.File.read("config/app.conf");

// 读取字节
byte[] raw = Vostok.File.readBytes("assets/logo.png");

目录操作

// 创建目录
Vostok.File.createDir("uploads/2024/01");

// 列出文件
List<String> files = Vostok.File.list("uploads");

// 递归列出
List<String> all = Vostok.File.listRecursive("uploads");

// 存在检查
boolean ok  = Vostok.File.exists("config/app.conf");
boolean dir = Vostok.File.isDirectory("uploads");

// 移动 / 重命名
Vostok.File.move("temp/draft.txt", "published/article.txt");

// 删除
Vostok.File.delete("temp/draft.txt");
Vostok.File.deleteDir("temp");

压缩与解压

// GZIP 压缩
Vostok.File.gzip("data/large.log", "data/large.log.gz");

// GZIP 解压
Vostok.File.ungzip("data/large.log.gz", "data/large.log");

// ZIP 解压（含 Zip Slip 防护）
Vostok.File.unzip("archives/data.zip", "output/");
安全
ZIP 解压内置 Zip Slip 路径穿越防护和 Zip Bomb 炸弹检测，通过 unzipMaxEntries / unzipMaxTotalUncompressedBytes 限制解压规模。

文件监听

// 监听目录变化（非递归）
Vostok.File.watch("config/", event -> {
    System.out.println("changed: " + event.path() + " type=" + event.type());
});

// 递归监听
Vostok.File.watch("uploads/", event -> { /* … */ }, true);

多存储后端

// 注册自定义存储实现
Vostok.File.registerStore("s3", new S3FileStore(s3Client, bucket));

// 切换默认后端
Vostok.File.setDefaultMode("s3");

// 临时使用指定后端
Vostok.File.withMode("s3", () -> {
    Vostok.File.write("remote/file.txt", data);
});

只读模式

Vostok.File.setReadOnly(true);   // 开启只读（写操作将抛出 FK-560）
Vostok.File.setReadOnly(false);  // 恢复可写
boolean ro = Vostok.File.isReadOnly();

配置参数

参数类型默认值说明
modeString"local"默认存储后端名称
baseDirStringuser.dir/vkfiles本地存储根目录
charsetString"UTF-8"文本编码
unzipMaxEntriesint1000ZIP 解压最大条目数
unzipMaxTotalUncompressedByteslong500MBZIP 解压总字节上限
unzipMaxEntryUncompressedByteslong200MB单条目解压字节上限
watchRecursiveDefaultbooleanfalsewatch 默认是否递归
datePartitionPatternString—日期分区目录格式（如 yyyy/MM/dd）

API 速查

方法说明
create(path, content)创建文件
write(path, content)覆盖写入
append(path, content)追加写入
read(path)读取文本
readBytes(path)读取字节
delete(path)删除文件
deleteDir(path)删除目录（递归）
exists(path)文件/目录是否存在
isDirectory(path)是否为目录
list(path)列出直接子文件/目录
listRecursive(path)递归列出所有文件
createDir(path)创建目录（含父目录）
move(from, to)移动/重命名
gzip(src, dest)GZIP 压缩
ungzip(src, dest)GZIP 解压
unzip(src, destDir)ZIP 解压
watch(path, handler)监听文件/目录变化
registerStore(mode, store)注册自定义存储后端
withMode(mode, action)临时切换存储后端
setReadOnly(flag)设置只读模式

异常处理

try {
    Vostok.File.read("missing.txt");
} catch (yueyang.vostok.file.exception.VKFileException e) {
    System.out.println(e.getCode() + " " + e.getMessage());
    // e.g. FK-404 路径不存在
}
