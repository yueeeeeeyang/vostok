🔗 Vostok.Http

Http 客户端模块

统一 HTTP 客户端，支持命名 Client、重试与退避、连接池、JSON 自动解析、流式响应、SSE 事件流、请求拦截器和指标统计。

初始化

Vostok.Http.init(new VKHttpConfig()
    .connectTimeoutMs(3000)
    .totalTimeoutMs(10000)
    .maxRetries(1)
    .retryBackoffBaseMs(200)
);

快速请求

// GET
VKHttpResponse res = Vostok.Http.get("https://api.example.com/users")
    .header("Accept", "application/json")
    .execute();
System.out.println(res.status() + " " + res.body());

// GET 并自动解析 JSON
User user = Vostok.Http.get("https://api.example.com/users/1")
    .executeJson(User.class);

// POST JSON
VKHttpResponse res = Vostok.Http.post("https://api.example.com/users")
    .bodyJson(user)
    .execute();

// 表单提交
VKHttpResponse res = Vostok.Http.post("https://example.com/form")
    .form(Map.of("username", "alice", "password", "secret"))
    .execute();

请求构建器

// 使用 put/patch/delete/head
VKHttpResponse res = Vostok.Http.put("https://api.example.com/users/1")
    .bearer(token)
    .query("dryRun", "true")
    .bodyJson(Map.of("name", "Bob"))
    .timeoutMs(5000)
    .retry(2)
    .execute();

命名 Client

// 注册带 baseUrl 的命名 Client
Vostok.Http.registerClient("payments", new VKHttpClientConfig()
    .baseUrl("https://payments.internal")
    .connectTimeoutMs(2000)
    .maxRetries(5)
    .defaultHeader("X-Api-Key", apiKey));

// 在命名 Client 上下文中使用相对路径
Vostok.Http.withClient("payments", () -> {
    VKHttpResponse res = Vostok.Http.post("/charge")
        .bodyText(chargeJson)
        .execute();
});

异步请求

// 异步执行，返回 CompletableFuture
CompletableFuture<VKHttpResponse> future =
    Vostok.Http.get("https://api.example.com/data").executeAsync();

future.thenAccept(res -> System.out.println(res.body()));

// 异步 JSON 解析
CompletableFuture<User> futureUser =
    Vostok.Http.get("https://api.example.com/users/1")
        .executeJsonAsync(User.class);

流式响应

// 分块接收大文件
Vostok.Http.get("https://files.example.com/large.zip")
    .executeStream((chunk, done) -> {
        outputStream.write(chunk);
        if (done) outputStream.close();
    });

SSE（Server-Sent Events）

// 接收 SSE 事件流
Vostok.Http.get("https://api.example.com/events")
    .header("Accept", "text/event-stream")
    .executeSse(new VKHttpSseListener() {
        public void onEvent(String event, String data, String id) {
            System.out.println("event=" + event + " data=" + data);
        }
        public void onError(Throwable ex) {}
        public void onComplete() {}
    });

// 保持长连接（返回 Session 对象，可手动关闭）
VKHttpStreamSession session = Vostok.Http.get(url)
    .openSse(listener);
// 稍后关闭
session.close();

拦截器

// 全局拦截器（请求/响应日志）
Vostok.Http.addInterceptor((req, chain) -> {
    Vostok.Log.debug("→ {} {}", req.method(), req.url());
    VKHttpResponse res = chain.proceed(req);
    Vostok.Log.debug("← {}", res.status());
    return res;
});

// 针对命名 Client 的拦截器
Vostok.Http.addInterceptor("payments", (req, chain) -> {
    return chain.proceed(req.withHeader("X-Trace", UUID.randomUUID().toString()));
});

指标

VKHttpMetrics m = Vostok.Http.metrics();
System.out.printf("totalReq=%d, errors=%d, avgMs=%.1f%n",
    m.totalRequests(), m.errorCount(), m.avgResponseMs());

Vostok.Http.metrics("payments");  // 命名 Client 指标
Vostok.Http.resetMetrics();

配置参数

参数类型默认值说明
connectTimeoutMslong3000TCP 连接超时（ms）
totalTimeoutMslong10000请求总超时（ms，0=无限制）
readTimeoutMslong0读取超时（0=无限制）
maxRetriesint1失败最大重试次数
retryBackoffBaseMslong100重试基础退避时间（指数退避）
failOnNon2xxbooleantrue非 2xx 状态码时是否抛异常
maxResponseBodyByteslong8388608响应体最大字节数（8MB）
metricsEnabledbooleantrue开启请求指标统计
requestBodyCompressionbooleanfalse自动 GZIP 压缩请求体

API 速查

方法说明
get(url) / post(url) / put(url) / delete(url)快捷构建请求
.execute()同步执行
.executeAsync()异步执行（CompletableFuture）
.executeJson(Class)同步执行并解析 JSON
.executeJsonAsync(Class)异步执行并解析 JSON
.executeStream(listener)流式响应
.executeSse(listener)SSE 事件流（完成后关闭）
.openSse(listener)SSE 持久连接（返回 Session）
registerClient(name, config)注册命名 HTTP Client
withClient(name, action)切换命名 Client 执行
addInterceptor(interceptor)全局拦截器
addInterceptor(name, interceptor)命名 Client 拦截器
metrics()全局请求指标
urlEncode(value)URL 编码工具
