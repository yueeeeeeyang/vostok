📋 Vostok.Log

Log 异步日志模块

高性能异步日志，支持文件滚动（按大小/时间）、压缩归档、MDC 上下文、命名 Logger、ANSI 彩色控制台输出，写入前经队列缓冲，不阻塞业务线程。

初始化

// 使用默认配置（INFO 级别，输出到 ./logs）
Vostok.Log.init();

// 自定义配置
Vostok.Log.init(new VKLogConfig()
    .level(VKLogLevel.DEBUG)
    .outputDir("logs")
    .filePrefix("app")
    .consoleEnabled(true)
    .consoleColor(true)
    .maxFileSizeBytes(100 * 1024 * 1024)  // 100 MB 滚动
    .maxBackups(10)
    .rollInterval(VKLogRollInterval.DAILY)
    .compressRolledFiles(true)
    .queueCapacity(65536)
);
懒加载
未显式调用 init() 时，首次写日志会自动以默认配置初始化，无需提前调用。

写日志

静态 API（自动推断调用者名称）

Vostok.Log.trace("entering method");
Vostok.Log.debug("config loaded: {}", configPath);
Vostok.Log.info("server started on port {}", port);
Vostok.Log.warn("slow query: {} ms", elapsed);
Vostok.Log.error("failed to connect", exception);

命名 Logger

// 获取命名 Logger
VKLogger log = Vostok.Log.logger("payment");

log.info("processing order {}", orderId);
log.warn("retry #{}", retryCount);
log.error("payment failed", ex);

MDC（映射诊断上下文）

// 设置 MDC，后续日志自动携带这些字段
Vostok.Log.mdc("requestId", requestId);
Vostok.Log.mdc("userId", userId);

Vostok.Log.info("handling request");  // 自动包含 requestId / userId

// 作用域 MDC（自动清理）
Vostok.Log.mdc(Map.of("traceId", traceId), () -> {
    Vostok.Log.info("inside scope");
});

// 手动清理
Vostok.Log.mdcClear();

动态调整日志级别

Vostok.Log.setLevel(VKLogLevel.DEBUG);  // 热切换，无需重启

刷盘

Vostok.Log.flush();     // 刷新当前默认 Logger 队列
Vostok.Log.flushAll();  // 刷新所有 Logger 队列

配置参数

参数类型默认值说明
levelVKLogLevelINFO全局日志级别（TRACE/DEBUG/INFO/WARN/ERROR）
outputDirString"logs"日志文件输出目录
filePrefixString"vostok"日志文件名前缀
maxFileSizeByteslong100MB单文件最大大小，超出后滚动
maxBackupsint10保留历史文件数量
maxBackupDaysint30历史文件保留天数
maxTotalSizeByteslong1GB历史文件总大小上限
rollIntervalVKLogRollIntervalDAILY按时间滚动：HOURLY / DAILY / WEEKLY
compressRolledFilesbooleanfalse归档文件异步 GZIP 压缩
consoleEnabledbooleanfalse同时输出到控制台
consoleColorbooleanfalse控制台 ANSI 彩色输出
queueCapacityint32768异步队列容量
queueFullPolicyVKLogQueueFullPolicyDROP队列满时策略：DROP / BLOCK / SYNC_FALLBACK
flushIntervalMslong500自动刷盘间隔（ms）
fsyncPolicyVKLogFsyncPolicyNEVERfsync 策略：NEVER / EVERY_FLUSH / EVERY_WRITE
formatterVKLogFormatter默认格式自定义日志格式化器
errorListenerVKLogErrorListener—ERROR 级别事件回调

API 速查

方法说明
trace(msg, args...)TRACE 级别日志
debug(msg, args...)DEBUG 级别日志
info(msg, args...)INFO 级别日志
warn(msg, args...)WARN 级别日志
error(msg, Throwable)ERROR 级别日志（含异常栈）
logger(name)获取命名 Logger 实例
mdc(key, value)设置 MDC 键值
mdc(map, Runnable)作用域 MDC
mdcClear()清理当前线程 MDC
setLevel(VKLogLevel)动态修改全局日志级别
flush()刷新默认 Logger 队列
flushAll()刷新所有 Logger 队列
close()关闭日志系统，等待队列清空
