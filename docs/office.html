📊 Vostok.Office

Office 文档处理模块

提供 Office 相关能力入口。当前版本聚焦 Excel（.xlsx）读写：支持全量读取、流式逐行读取、生成工作簿，并复用 File 模块的 ZIP/UNZIP 能力与 Security 模块的安全检测链路。

初始化（建议显式）

import yueyang.vostok.Vostok;
import yueyang.vostok.file.VKFileConfig;
import yueyang.vostok.office.VKOfficeConfig;

// Office 依赖 File 模块（用于本地文件读写与 zip/unzip）
Vostok.File.init(new VKFileConfig().baseDir("./data"));

// Office 配置（可选）
Vostok.Office.init(new VKOfficeConfig()
    .excelMaxSheets(256)
    .excelMaxRowsPerSheet(1_000_000)
    .excelMaxColsPerRow(16_384)
    .excelMaxCellChars(32_767)
    .excelMaxWorkbookBytes(512L * 1024 * 1024)
    .excelTempDir("tmp/excel")
);
当前限制
仅支持 .xlsx，且目前只支持 Vostok.File 的 local 模式。

写入 Excel

import yueyang.vostok.office.excel.*;

VKExcelWorkbook workbook = new VKExcelWorkbook()
    .addSheet(new VKExcelSheet("Orders")
        .addCell(VKExcelCell.stringCell(1, 1, "orderId"))
        .addCell(VKExcelCell.stringCell(1, 2, "amount"))
        .addCell(VKExcelCell.stringCell(2, 1, "A001"))
        .addCell(VKExcelCell.numberCell(2, 2, "99.5"))
        .addCell(VKExcelCell.formulaCell(3, 2, "SUM(B2:B2)", "99.5")));

Vostok.Office.writeExcel("excel/orders.xlsx", workbook);

读取 Excel（全量）

import yueyang.vostok.office.excel.VKExcelWorkbook;

VKExcelWorkbook wb = Vostok.Office.readExcel("excel/orders.xlsx");
String orderId = wb.getSheet("Orders").getCell(2, 1).value();

读取 Excel（流式逐行，推荐大文件）

import yueyang.vostok.office.excel.VKExcelReadOptions;

Vostok.Office.readExcelRows(
    "excel/orders.xlsx",
    "Orders",
    VKExcelReadOptions.defaults().maxRowsPerSheet(1_000_000),
    row -> {
        int rowNo = row.rowIndex();
        String id = row.cells().containsKey(1) ? row.cells().get(1).value() : null;
        process(rowNo, id);
    }
);
内存建议
readExcelRows() 采用流式 StAX 解析，适合大文件导入；readExcel() 会构建完整工作簿对象，建议用于中小文件。

安全链路

  • 路径安全：调用 Vostok.Security.assertSafePath 检查输入路径。
  • 文件识别：读取文件头并校验 ZIP 魔数，拒绝伪装文件。
  • 脚本伪装：调用 checkExecutableScriptUpload 检测恶意上传。
  • XML 安全：禁用 DTD/外部实体，并做 XXE 样本检测。
  • 解压安全：复用 File 模块 unzip 限制（Zip Slip / Zip Bomb 防护）。

API 速查

方法说明
init()使用默认配置初始化 Office 模块
init(config)按指定配置初始化 Office 模块
readExcel(path)读取 .xlsx（全量）
readExcel(path, options)带限制选项读取 .xlsx（全量）
writeExcel(path, workbook)写入 .xlsx
writeExcel(path, workbook, options)带限制选项写入 .xlsx
readExcelRows(path, sheetName, options, consumer)按行流式读取 .xlsx
started()是否已初始化
config()获取当前配置副本
close()关闭 Office 模块
