📊 Vostok.Office

Office 文档处理模块

提供 Office 相关能力入口。当前支持 Excel（.xlsx）与 Word（.docx）能力：Excel 读写/流式读取，Word 文本与图片读取、字数与图片数统计，以及文本+图片的 Word 生成。

初始化（建议显式）

import yueyang.vostok.Vostok;
import yueyang.vostok.file.VKFileConfig;
import yueyang.vostok.office.VKOfficeConfig;

// Office 依赖 File 模块（用于本地文件读写与 zip/unzip）
Vostok.File.init(new VKFileConfig().baseDir("./data"));

Vostok.Office.init(new VKOfficeConfig()
    .excelTempDir("tmp/excel")
    .wordTempDir("tmp/word")
    .wordMaxTextChars(20_000_000)
    .wordMaxImages(10_000));
当前限制
仅支持 .xlsx 与 .docx，且目前只支持 Vostok.File 的 local 模式。

Excel 示例

import yueyang.vostok.office.excel.*;

VKExcelWorkbook workbook = new VKExcelWorkbook()
    .addSheet(new VKExcelSheet("Orders")
        .addCell(VKExcelCell.stringCell(1, 1, "orderId"))
        .addCell(VKExcelCell.numberCell(2, 2, "99.5")));

Vostok.Office.writeExcel("excel/orders.xlsx", workbook);

Vostok.Office.readExcelRows("excel/orders.xlsx", "Orders",
    VKExcelReadOptions.defaults(), row -> process(row.rowIndex(), row.cells()));

Word 读取

import yueyang.vostok.office.word.*;

// 读取全文文本
String text = Vostok.Office.readWordText("word/orders.docx");

// 统计字数（仅计数路径，不构建全文字符串）
int chars = Vostok.Office.countWordChars("word/orders.docx");

// 读取全部图片（默认返回 bytes）
List<VKWordImage> images = Vostok.Office.readWordImages("word/orders.docx");

// 大文件模式：仅读取图片 metadata，避免 OOM
VKWordReadOptions metadataOnly = VKWordReadOptions.defaults()
    .imageLoadMode(VKWordImageLoadMode.METADATA_ONLY);
List<VKWordImage> metaImages = Vostok.Office.readWordImages("word/orders.docx", metadataOnly);

// 一次解包聚合读取：文本 + 字数 + 图片 + 图片数
VKWordDocument doc = Vostok.Office.readWord("word/orders.docx", metadataOnly);

Word 生成（文本 + 图片）

import yueyang.vostok.office.word.*;

VKWordWriteRequest request = new VKWordWriteRequest()
    .addParagraph("订单 A001")
    .addImageBytes("logo.png", logoBytes)   // bytes 输入
    .addImageFile("images/sign.png");       // 文件路径输入（相对 File.baseDir）

Vostok.Office.writeWord("word/orders.docx", request,
    VKWordWriteOptions.defaults()
        .maxImages(100)
        .maxSingleImageBytes(5L * 1024 * 1024));
字数统计规则
Word 字数按非空白 Unicode code point 统计；空格、换行、制表符不计入字数。

API 速查

方法说明
readExcel(path)读取 .xlsx（全量）
writeExcel(path, workbook)写入 .xlsx
readExcelRows(path, sheetName, options, consumer)流式逐行读取 .xlsx
readWordText(path)读取 .docx 全文文本
readWordImages(path, options)读取 .docx 全部图片（支持 metadata only）
countWordChars(path, options)统计 .docx 字数（仅计数路径）
countWordImages(path, options)统计 .docx 图片数
readWord(path, options)一次解包聚合读取 Word 文档信息
writeWord(path, request, options)生成 .docx（文本 + 图片）
started()是否已初始化
config()获取当前配置副本
close()关闭 Office 模块
