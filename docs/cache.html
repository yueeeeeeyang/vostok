⚡ Vostok.Cache

Cache 缓存模块

统一缓存访问层，支持内存（Memory）、Redis 及分层（Tiered）三种后端，提供布隆过滤器、Pipeline 批量、命名缓存、命中率统计等能力。

初始化

内存缓存

Vostok.Cache.init(new VKCacheConfig()
    .providerType("MEMORY")
    .maxEntries(50000)
    .evictionPolicy(VKEvictionPolicy.LRU)
    .defaultTtlMs(3600_000)   // 1 小时
);

Redis 缓存

Vostok.Cache.init(new VKCacheConfig()
    .providerType("REDIS")
    .endpoints("127.0.0.1:6379")
    .password("yourpass")
    .database(0)
    .maxActive(20)
    .defaultTtlMs(86400_000)
);

分层缓存（L1 内存 + L2 Redis）

Vostok.Cache.init(new VKCacheConfig()
    .providerType("TIERED")
    .l1Config(new VKCacheConfig()
        .providerType("MEMORY").maxEntries(1000).defaultTtlMs(60_000))
    .l2Config(new VKCacheConfig()
        .providerType("REDIS").endpoints("127.0.0.1:6379").defaultTtlMs(3600_000))
);
说明
Cache 模块不依赖 Config 模块，必须通过 VKCacheConfig 显式初始化，不支持懒加载。

基本操作

写入 / 读取

// 写入（使用默认 TTL）
Vostok.Cache.set("user:1", user);

// 写入（自定义 TTL，单位 ms）
Vostok.Cache.set("token:abc", session, 1800_000L);

// 读取（返回 Object）
Object raw = Vostok.Cache.get("user:1");

// 读取（指定类型）
User user = Vostok.Cache.get("user:1", User.class);

// 懒加载：未命中时自动调用 loader 并缓存结果
User user = Vostok.Cache.getOrLoad("user:1", User.class, 3600_000, () -> {
    return Vostok.Data.findById(User.class, 1L);
});

删除 / 检查 / 过期

Vostok.Cache.delete("user:1", "user:2");
boolean exists = Vostok.Cache.exists("user:1");
Vostok.Cache.expire("user:1", 600_000L);  // 重置 TTL

计数器

long v  = Vostok.Cache.incr("pv:today");       // 加 1
long v2 = Vostok.Cache.incrBy("pv:today", 5);  // 加 N
Vostok.Cache.decr("stock:100");
Vostok.Cache.decrBy("stock:100", 3);

Pipeline 批量操作

// 批量写入，无需关心返回值
Vostok.Cache.pipeline(pipe -> {
    pipe.set("k1", "v1");
    pipe.set("k2", "v2");
    pipe.expire("k1", 60_000);
});

// 批量操作并获取结果
VKCachePipelineResult result = Vostok.Cache.pipelineWithResult(pipe -> {
    pipe.get("k1");
    pipe.get("k2");
});
String v1 = result.get(0, String.class);
String v2 = result.get(1, String.class);

命名缓存（多缓存实例）

// 注册额外命名缓存
Vostok.Cache.registerCache("session", new VKCacheConfig()
    .providerType("REDIS")
    .endpoints("session-redis:6379")
    .defaultTtlMs(1800_000));

// 切换到命名缓存
Vostok.Cache.withCache("session", () -> {
    Vostok.Cache.set("sid:abc", sessionObj);
});

Key 级锁

// 对同一 key 的并发操作串行化
String result = Vostok.Cache.withKeyLock("lock:order:42", () -> {
    Order order = Vostok.Cache.get("order:42", Order.class);
    order.setStatus("paid");
    Vostok.Cache.set("order:42", order);
    return order.getId();
});

命中率统计

VKCacheStats stats = Vostok.Cache.stats();
System.out.printf("hits=%d, misses=%d, hitRate=%.2f%%%n",
    stats.hits(), stats.misses(), stats.hitRate() * 100);

// 命名缓存统计
VKCacheStats s2 = Vostok.Cache.stats("session");
Vostok.Cache.resetStats();

事件监听

VKCacheConfig cfg = new VKCacheConfig()
    .eventListener((event) -> {
        if (event.type() == VKCacheEventType.EVICTED) {
            System.out.println("evicted: " + event.key());
        }
    });

自定义编解码器

Vostok.Cache.registerCodec(new VKCacheCodec() {
    public byte[] encode(Object value) { /* … */ }
    public Object decode(byte[] bytes, Class<?> type) { /* … */ }
});

配置参数

参数类型默认值说明
providerTypeString"MEMORY"后端类型：MEMORY / REDIS / TIERED
endpointsString—Redis 地址，逗号分隔
redisModeString"SINGLE"SINGLE / SENTINEL / CLUSTER
passwordString—Redis 密码
databaseint0Redis 数据库编号
sslbooleanfalse启用 TLS 连接
defaultTtlMslong-1默认 TTL（-1 永不过期）
keyPrefixString""Key 前缀，统一隔离命名空间
maxEntriesint10000内存最大条目数
evictionPolicyVKEvictionPolicyLRU淘汰策略：LRU / LFU / FIFO / NONE
minIdleint2Redis 连接池最小空闲
maxActiveint10Redis 连接池最大活跃
metricsEnabledbooleantrue开启命中率统计
nullCacheEnabledbooleanfalse缓存 null 值（防缓存穿透）
rateLimitQpsint0Redis 访问 QPS 限制（0=关闭）
l1ConfigVKCacheConfig—TIERED 模式 L1 配置
l2ConfigVKCacheConfig—TIERED 模式 L2 配置

API 速查

方法说明
set(key, value)写入（默认 TTL）
set(key, value, ttlMs)写入（自定义 TTL）
get(key)读取
get(key, Class)读取并转换类型
getOrLoad(key, Class, ttlMs, Supplier)读取或懒加载
delete(keys...)删除一或多个 key
exists(key)判断 key 是否存在
expire(key, ttlMs)重置过期时间
incr(key)原子自增 1
incrBy(key, delta)原子自增 N
decr(key)原子自减 1
pipeline(Consumer)Pipeline 批量（无结果）
pipelineWithResult(Consumer)Pipeline 批量（含结果）
withKeyLock(key, Supplier)Key 级别互斥锁
stats()获取命中率统计
resetStats()重置统计
registerCache(name, config)注册命名缓存
withCache(name, action)切换命名缓存执行
registerCodec(codec)注册自定义编解码器
poolMetrics()Redis 连接池指标
